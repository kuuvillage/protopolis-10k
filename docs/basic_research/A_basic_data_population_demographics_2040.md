# A. 基本データ収集：人口動態と自治体の変容（2040年構想）

調査日：2025年11月12日

## 1. 総人口の推移と将来推計

### 1.1 全国人口の長期推計

国立社会保障・人口問題研究所（IPSS）による令和5年（2023年）推計に基づく主要年次の人口推計は以下の通りである：

| 年次 | 総人口 | 基準年との差 | 備考 |
|------|--------|-------------|------|
| **2020年** | 1億2,615万人 | － | 推計基準年（令和2年10月1日現在） |
| **2030年** | 約1億1,284万人 | ▲1,331万人 | |
| **2040年** | 約1億0,469万人 | ▲2,146万人 | |
| **2050年** | 約9,492万人 | ▲3,123万人 | |
| **2070年** | 約8,700万人 | ▲3,915万人 | |

**出典：** 国立社会保障・人口問題研究所「日本の将来推計人口（令和5年推計）」

### 1.2 主要な推計の特徴

- **推計手法：** コーホート要因法（国際標準手法）
- **推計期間：** 2021年～2070年（50年間）
- **2100年の展望：** 現在の総人口の約半分程度まで減少する見込み
- **減少ペース：** 2020年から2040年の20年間で約2,100万人（約17%）減少

### 1.3 都道府県別の動向

2040年時点では全都道府県で人口減少が予測されている：

- **減少率が最も大きい地域：** 秋田県（約35%減少）
- **減少率が最も小さい地域：** 東京都（約6.5%減少）
- **地方圏全般：** 中山間地域を中心に20～30%以上の減少が見込まれる

## 2. 生産年齢人口と高齢人口の推移

### 2.1 生産年齢人口（15～64歳）の減少

| 年次 | 人口（万人） | 総人口比 | 2020年との差 |
|------|------------|---------|-------------|
| **2020年** | 7,406 | 59.1% | － |
| **2030年** | 6,875 | 約61.0% | ▲531万人 |
| **2040年** | 5,978 | 53.9% | ▲1,428万人 |
| **2050年** | 5,276 | 約55.6% | ▲2,130万人 |

**出典：** 内閣府「税制調査会資料」、総務省「情報通信白書」

### 2.2 高齢人口（65歳以上）の増加と高齢化率

| 年次 | 高齢化率 | 特記事項 |
|------|---------|---------|
| **2020年** | 約28.6% | 既に「超高齢社会」（21%超） |
| **2040年** | **約35.0%** | **団塊ジュニア世代が65歳到達** |
| **2065年** | 約39～40% | 高齢化率がピークに近づく |

### 2.3 2040年問題の核心

- **労働力人口の激減：** 生産年齢人口が20年間で約1,400万人（約19%）減少
- **平均年齢の上昇：** 年平均約73万人のペースで労働力が減少
- **社会保障負担の増大：** 現役世代1.5人が高齢者1人を支える構造へ

## 3. 自治体（市町村）数の変化と合併動向

### 3.1 平成の大合併とその影響

| 時期 | 市町村数 | 変化 | 備考 |
|------|---------|------|------|
| **1999年3月** | 3,229 | － | 平成の大合併開始前 |
| **2004年** | － | ▲249 | 合併ピーク手前 |
| **2005年** | － | **▲778** | **合併が最も進んだ年** |
| **2006年** | － | ▲326 | 合併第二のピーク |
| **2006年以降** | 1,821 | 約50%減 | 合併がほぼ完了 |
| **2024年現在** | 約1,718 | 微減継続 | 大規模合併は終息 |

**出典：** 総務省「市町村合併資料集」、全国市町村数変遷データ

### 3.2 合併の影響と現状

- **市の増加：** 町村が市に昇格するケースが増加
- **町村の減少：** 特に過疎地域の小規模町村が大幅に減少
- **指定都市・中核市の増加：** 都市機能の集約が進展
- **2020年代の動向：** 大規模合併は一段落し、個別の小規模合併のみ

### 3.3 今後の見通し

- **自発的な合併：** 財政難や人口減少を理由とした合併の可能性
- **連携協約・広域連合：** 合併によらない自治体間連携の模索
- **小規模自治体の課題：** 人口1万人未満の自治体の持続可能性が焦点

## 4. 地域別人口減少率と若年女性人口の動向

### 4.1 女性20～39歳人口の減少見通し

日本創成会議（Japan Policy Council）の分析による「消滅可能性都市」の指標：

**2040年までに女性20～39歳人口が50%以上減少する自治体：**

| カテゴリ | 自治体数 | 割合 |
|---------|---------|------|
| **全自治体** | 896 | **49.8%** |
| 人口1万人以上 | 373 | － |
| 人口1万人未満 | 523 | － |

**出典：** 日本創成会議「全国市区町村別『20～39歳女性』の将来推計人口」

### 4.2 地域別の深刻度

**最も深刻な地域：**

- **秋田県：** 大潟村を除く全自治体で50%以上減少
- **青森県：** 約90%の自治体で50%以上減少
- **その他東北・北陸・四国・九州の中山間地域**

**都市部でも深刻化：**

- **東京23区：** 豊島区など一部区で50%超減少
- **減少幅：** 23区内でも10.2%～50.8%とばらつき
- **大阪・神戸：** 一部区で若年女性人口が半減

### 4.3 若年女性人口減少の意味

- **出生率への影響：** 次世代の出生数がさらに減少
- **地域経済への影響：** 消費・サービス業の縮小
- **コミュニティの持続性：** 学校・保育所などの存続困難
- **地域間格差の拡大：** 都市部への一極集中がさらに加速

##5. 移住・転入転出データと外国人住民の動向

### 5.1 主要データソース

総務省統計局が公表する以下のデータが利用可能：

1. **住民基本台帳人口移動報告**
   - 月次・年次で転入転出のデータを集計
   - URL: https://www.stat.go.jp/data/idou/index.html

2. **外国人の国内移動の状況（2022年特別報告）**
   - 外国人住民の都道府県間・市町村間移動を分析
   - 国籍別・年齢別・地域別の詳細データを含む

3. **e-Stat（政府統計の総合窓口）**
   - データベース検索による詳細な時系列データの取得が可能
   - URL: https://www.e-stat.go.jp/

### 5.2 近年の移住動向（概要）

**転入超過の傾向：**

- **東京圏（東京・埼玉・千葉・神奈川）：** 一貫して転入超過
- **大阪圏・名古屋圏：** 転入超過と転出超過が年により変動
- **地方圏：** 大部分が転出超過（特に若年層）

**コロナ禍以降の変化：**

- 2020～2021年：東京圏への転入超過が一時的に縮小
- 2022年以降：東京圏への転入超過が再拡大傾向
- テレワーク普及による地方移住は限定的

### 5.3 外国人住民の動向

**全体傾向：**

- **外国人人口：** 2020年代に入り増加傾向が継続（コロナ禍で一時減少後、回復）
- **主な国籍：** 中国、韓国、ベトナム、フィリピン、ブラジルなど
- **居住地域：** 都市部および製造業集積地域に集中

**地方における外国人住民：**

- **技能実習生・特定技能外国人：** 地方の農業・製造業・介護分野で増加
- **定住傾向：** 家族帯同や永住者が増加し、地域社会への統合が課題
- **外国人比率：** 一部自治体では10%超の地域も出現

### 5.4 2040年に向けた示唆

- **日本人人口の減少補完：** 外国人住民が地域の人口・労働力の一定割合を担う可能性
- **多文化共生の必要性：** 教育・行政サービスの多言語対応が不可欠
- **地域コミュニティの変容：** 外国人住民を包摂する新しい自治体モデルが求められる

## 6. 調査のまとめと含意

### 6.1 主要な発見

1. **人口の急激な減少：** 2020年から2040年の20年間で約2,100万人（17%）減少
2. **生産年齢人口の激減：** 同期間に約1,400万人（19%）減少
3. **高齢化の加速：** 2040年には国民の3人に1人以上が65歳以上
4. **自治体の半数が危機：** 約50%の自治体で若年女性人口が半減
5. **外国人住民の増加：** 人口減少を補完する重要な要素となる可能性

### 6.2 人口1万人未満の自治体への示唆

- **存続の危機：** 多くの小規模自治体が若年層の流出により持続困難に
- **サービス維持の困難：** 行政・医療・教育の基盤維持が課題
- **新しいモデルの必要性：** 従来型の自治体運営からの転換が不可避

### 6.3 次のステップ

本基本データを踏まえ、以下の調査を進める必要がある：

- **B. 地域・自治体機能：** 小規模自治体の財政構造と行政サービスの実態
- **C. 経済・技術・労働：** 労働力不足への対応と技術導入の可能性
- **D. 社会・暮らし・文化：** コミュニティの持続と新しい参加モデル
- **E. 2040年シナリオ設計：** 複数の未来シナリオの構築と評価
