# B. 地域・自治体機能：小規模自治体の実態と課題（2040年構想）

調査日：2025年11月12日

## 1. 小規模自治体の現状と将来見通し

### 1.1 小規模自治体の定義と規模

**人口規模別の自治体分類：**

| 人口規模 | 自治体数（概算） | 割合 | 特徴 |
|---------|---------------|------|------|
| 5,000人未満 | 約180 | 約10% | 最小規模・行政機能維持が困難 |
| 5,000～10,000人 | 約200 | 約12% | 小規模・基礎的サービス維持が課題 |
| 10,000～50,000人 | 約600 | 約35% | 中小規模・地方中核機能が必要 |
| 50,000人以上 | 約740 | 約43% | 中核都市・広域サービス提供可能 |

**出典：** 総務省「小規模市町村の状況」、国土交通省資料

### 1.2 過疎地域と消滅可能性都市

**全国の過疎地域の現状：**

- **過疎市町村数：** 744自治体（全1,729自治体の約43%）
- **消滅可能性都市：** 896自治体（全体の約52%）
- **人口割合：** 全国人口の9.2%が過疎地域に居住
- **面積割合：** 国土面積の63.2%を過疎地域が占める
- **高齢化率：** 過疎地域では39.7%（全国平均を約11ポイント上回る）

**出典：** 総務省「過疎対策」、全国過疎地域連盟

### 1.3 人口規模別の実態分析

**人口5,000～10,000人規模の自治体の特徴：**

1. **地理的特性：**
   - 大部分が中山間地域・離島・半島部に立地
   - 都市部へのアクセスに時間を要する地域が多い

2. **人口動態：**
   - 年平均1～3%の人口減少が継続
   - 若年層（20～39歳）の流出が顕著
   - 2040年までに現在の人口の30～50%減少が予測される

3. **行政体制：**
   - 職員数：30～100名程度
   - 専門職（技術職・福祉職）の確保が困難
   - 広域連携や事務委託に依存

### 1.4 成功事例：人口規模を維持してきた自治体

衆議院調査局の2023年実証研究によれば、一部の小規模自治体は人口を安定的に維持している：

**成功要因：**

- **産業基盤の維持：** 地域固有の産業（農林水産業、観光、製造業）の活性化
- **移住政策の充実：** 住宅支援、就労支援、子育て支援の総合的提供
- **交通・ICTインフラ：** アクセス改善とデジタル環境の整備
- **地域コミュニティの活性：** 住民参加型のまちづくり

**代表事例：**

- **徳島県神山町（人口約5,000人）：**
  - 高速ブロードバンド整備とサテライトオフィス誘致
  - 40社以上が拠点を構え、2016年には156世帯234名が移住
  - ICTを活用した地域活性化のモデルケース

## 2. 自治体財政の構造と小規模自治体の財政状況

### 2.1 地方財政の基本構造

**歳入の主要構成要素：**

1. **自主財源：**
   - 地方税（住民税・固定資産税など）
   - 使用料・手数料

2. **依存財源：**
   - 地方交付税交付金
   - 国庫支出金
   - 都道府県支出金
   - 地方債

### 2.2 地方交付税交付金の仕組み

**地方交付税交付金とは：**

「国が地方に代わって徴収する税」とも表現され、地方自治体間の財源の不均衡を調整し、どの地域でも一定水準の行政サービスを提供できるようにする制度である。

**財源：**

- 所得税の33.1%
- 法人税の33.1%
- 酒税の50%
- 消費税の19.5%
- 地方法人税の全額

**交付の種類：**

1. **普通交付税（94%）：**
   - 計算式：基準財政需要額 － 基準財政収入額 = 交付額
   - 全自治体の約9割が対象

2. **特別交付税（6%）：**
   - 災害など特別の財政需要に対応

**出典：** 総務省「地方財政制度」

### 2.3 財政力指数と自治体の財政状況

**財政力指数とは：**

$$
\text{財政力指数} = \frac{\text{基準財政収入額}}{\text{基準財政需要額}}
$$

- **指数1.0以上：** 地方交付税の交付を受けない「不交付団体」
- **指数1.0未満：** 地方交付税の交付を受ける「交付団体」
- **指数0.3以下：** 財政基盤が極めて脆弱

**小規模自治体の財政力指数：**

| 人口規模 | 平均財政力指数 | 交付税依存度 | 特徴 |
|---------|--------------|------------|------|
| **5,000人未満** | 0.15～0.30 | 60～80% | 極めて脆弱・依存度が非常に高い |
| **5,000～10,000人** | 0.25～0.40 | 50～70% | 脆弱・地方交付税に大きく依存 |
| **10,000～50,000人** | 0.30～0.55 | 40～60% | 中程度・地域差が大きい |
| **50,000人以上** | 0.50～1.20 | 20～50% | 比較的安定 |

**出典：** 総務省「地方公共団体の主要財政指標一覧」、各都道府県公表データ

### 2.4 小規模自治体の財政上の課題

1. **歳入の脆弱性：**
   - 自主財源（地方税）が極めて少ない
   - 地方交付税への依存度が60～80%に達する
   - 人口減少により税収がさらに減少

2. **歳出の硬直性：**
   - 固定的経費（人件費・公債費）の割合が高い
   - スケールメリットが働かず、住民一人あたりの行政コストが高い
   - インフラ維持管理費が重い負担

3. **財政調整基金の枯渇：**
   - 災害や緊急事態への対応力が低下
   - 新規事業への投資が困難

4. **2040年に向けたリスク：**
   - 人口減少による税収減
   - 高齢化による社会保障費の増大
   - インフラ老朽化に伴う更新需要の増大
   - 国の財政悪化による地方交付税の削減リスク

## 3. 地域サービスの現状と課題（医療・交通・教育・ICT）

### 3.1 医療・介護サービスの課題

**2040年問題における医療・介護の危機：**

| 項目 | 現状（2020年代） | 2040年の見通し |
|------|----------------|---------------|
| **高齢者人口** | 約3,600万人 | **約4,000万人（ピーク）** |
| **85歳以上人口** | 約600万人 | **約1,000万人** |
| **認知症患者** | 約600万人 | **約950万人** |
| **医療・介護人材** | 不足が顕在化 | **深刻な人手不足** |

**出典：** 厚生労働省、全日本病院協会「病院のあり方に関する報告書（2021年版）」

**小規模自治体における医療・介護の課題：**

1. **医療機関の不足・撤退：**
   - 診療所・病院の医師確保が困難
   - 救急医療体制の維持が困難
   - 専門医療へのアクセスに数時間を要する

2. **介護サービスの限界：**
   - 訪問介護・デイサービスの事業者不足
   - 介護施設の定員不足
   - 介護職員の確保が困難

3. **2040年への対応策：**
   - **遠隔医療（オンライン診療）の普及**
   - **医療・介護連携の強化（地域医療構想）**
   - **健康寿命の延伸施策**
   - **AI・ロボット技術の導入**

### 3.2 交通インフラの課題

**小規模自治体における交通の現状：**

- **公共交通の衰退：** 鉄道・バス路線の廃止・減便
- **高齢者の移動困難：** 自家用車に依存、運転免許返納後の移動手段がない
- **物流の非効率性：** 過疎地域への配送コスト増大

**2040年に向けた期待と課題：**

| 技術・施策 | 期待される効果 | 課題 |
|-----------|--------------|------|
| **自動運転** | 高齢者や障害者の移動支援 | インフラ整備、法規制、採算性 |
| **デマンド交通** | 効率的な公共交通 | 運用コスト、利用者の習熟 |
| **ドローン配送** | 物流の効率化 | 法規制、安全性、天候依存 |
| **MaaS** | シームレスな移動 | 地方での事業採算性 |

**出典：** 国土交通省「地域公共交通の活性化・再生への事例集」

### 3.3 教育サービスの課題

**小規模自治体における教育の現状：**

1. **学校の統廃合：**
   - 少子化により小中学校の統廃合が進行
   - 通学距離の増大、スクールバスの負担

2. **教員確保の困難：**
   - 小規模校では専門教員の配置が困難
   - 教員の高齢化と若手の不足

3. **教育の質の確保：**
   - 多様な学習機会の提供が困難
   - 部活動・課外活動の選択肢が限定的

**ICTを活用した教育の可能性：**

- **遠隔授業：** 専門教員による授業の配信
- **個別最適化学習：** AIによる習熟度別指導
- **教育クラウドプラットフォーム：** 学習教材の共有と活用
- **ICT支援員の配置：** 教員の負担軽減とICT活用の促進

**出典：** 文部科学省資料

### 3.4 ICTインフラとデジタル化の課題

**自治体のICT化の現状：**

1. **進展している分野：**
   - 住民票・証明書のオンライン申請
   - マイナンバーカードの活用
   - 行政手続きのデジタル化（e-Gov）

2. **遅れている分野：**
   - 行政業務の内部システムの統合・効率化
   - データ連携・オープンデータの活用
   - AI・RPA（ロボティック・プロセス・オートメーション）の導入

**ICT化の課題（特に小規模自治体）：**

| 課題 | 詳細 |
|------|------|
| **導入・運用コストが高い** | 初期投資と維持管理費用が財政を圧迫 |
| **人材・ノウハウ不足** | ICT専門職員がいない、外部委託に依存 |
| **費用対効果が不明確** | 小規模自治体では効果が限定的な場合も |
| **セキュリティリスク** | サイバー攻撃への対策が不十分 |
| **デジタルデバイド** | 高齢者など一部住民がICTを利用できない |

**出典：** 総務省「ICT地域活性化事例100選」、自治体通信Online

**成功事例：ICTを活用した自治体の取り組み：**

1. **神山町（徳島県）：**
   - 高速ブロードバンド整備により、サテライトオフィス誘致に成功

2. **会津若松市（福島県）：**
   - スーパーシティ構想による包括的なICT活用

3. **加賀市（石川県）：**
   - 行政手続きの完全デジタル化とAI活用

### 3.5 ライフライン・都市機能の維持課題

**インフラ維持の困難：**

- **水道・電気・ガス：** 利用者減少による採算性悪化
- **道路・橋梁：** 老朽化と更新費用の増大
- **公共施設：** 統廃合と長寿命化の必要性
- **防災機能：** 消防・救急体制の広域化

**2040年に向けた対応：**

- **公共施設等総合管理計画：** 施設の統廃合と効率化
- **PPP/PFI：** 民間資金・ノウハウの活用
- **広域連携：** 複数自治体での共同運営

## 4. 過疎地域の持続可能性と法的枠組み

### 4.1 過疎地域持続的発展支援法

令和3年度（2021年）から「過疎地域の持続的発展の支援に関する特別措置法」が施行されている。

**法律の概要：**

- **施行期間：** 令和3年4月1日～令和13年（2031年）3月31日（10年間の時限立法）
- **対象地域：** 過疎地域として指定された市町村
- **支援内容：**
  - 過疎対策事業債（過疎債）による財政支援
  - 税制上の優遇措置
  - 国の補助率のかさ上げ

**出典：** 総務省「過疎対策」、e-Gov法令検索

### 4.2 過疎地域の持続可能性への課題

**人口減少による影響：**

- **ライフライン維持困難：** 水道、電気、ガスの供給維持
- **公共交通の消滅：** 移動手段の喪失
- **医療・介護の空白地帯化：** 高齢者の生活維持が困難
- **防災力の低下：** 消防団員の減少、災害時の共助機能の低下
- **地域活動の衰退：** 祭りや伝統行事の継承困難

**2040年に向けた持続可能性のカギ：**

1. **定住人口の確保：**
   - 移住促進策の強化
   - 雇用の場の創出

2. **関係人口の拡大：**
   - 二拠点居住・ワーケーションの促進
   - ふるさと納税・地域貢献の仕組み

3. **デジタル技術の活用：**
   - 遠隔医療・教育・行政サービス
   - スマート農業・林業

4. **広域連携の強化：**
   - 自治体間の事務共同処理
   - 連携中枢都市圏・定住自立圏の形成

## 5. 調査のまとめと含意

### 5.1 主要な発見

1. **小規模自治体の厳しい現実：**
   - 全自治体の約半数が過疎問題を抱える
   - 人口5,000～10,000人規模の自治体は財政基盤が極めて脆弱
   - 地方交付税への依存度が60～80%に達する

2. **地域サービス維持の困難：**
   - 医療・介護：2040年には高齢者4,000万人、介護人材不足が深刻化
   - 交通：公共交通の衰退、自動運転などの新技術への期待
   - 教育：学校統廃合、ICT活用による質の確保が課題
   - ICT：導入コスト・人材不足が障壁

3. **成功事例の存在：**
   - 神山町などICTを活用した地域活性化の成功例
   - 移住促進・産業振興により人口を維持する自治体も存在

### 5.2 人口1万人未満の自治体への示唆

**従来型自治体運営の限界：**

- 財政基盤の脆弱性により、単独での行政サービス維持が困難
- 専門職員の確保が不可能、住民一人あたり行政コストが高い
- インフラ老朽化への対応が財政を圧迫

**新しい自治体モデルの必要性：**

- **デジタル技術の徹底活用：** 遠隔サービス、行政効率化
- **広域連携の強化：** 複数自治体での機能分担
- **住民参加の促進：** 共助・自助の仕組みづくり
- **外部人材の活用：** 関係人口、外国人住民の包摂

### 5.3 次のステップ

本調査結果を踏まえ、以下の調査を進める必要がある：

- **C. 経済・技術・労働：** 地域経済の構造と技術導入の可能性
- **D. 社会・暮らし・文化：** コミュニティの実態と新しい参加モデル
- **E. 2040年シナリオ設計：** 複数の未来シナリオと対応策の検討
